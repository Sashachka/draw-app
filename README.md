# draw-app
My Study Projects. Draw app, from University of London

here is the site:
https://sashachka.github.io/draw-app/draw-app.coursework/

Canvas has two layers. Image can be saved or completely cleared(on first and second layer). And undo/redo(undo deletes last action, redo makes that last action appear again) buttons at the bottom. 

Colours are changeable on each of the functions

Pencil, line and spray bottle has changeable colours with gui librarry.

Mirror pencil has button to make the line of mirror horizontal or vise versa vertical.

Star and flower stamps has changeable size, number of stars and changeable number of points/petals.

Both of the trees and bee has changeable size and number of them. But bees cannot cahnge colour.

Edit shape with pretty same icon as line but has different usage. It draws a line after clicking button edit shape it shows red dots all over line each of them you can drag in different directions to change it's shape after doing don't forget to click finish shape it will save the drawing and will not be affected after again clicking edit shape.

Circle and rectangle shapes allows you to draw different shaes of circle and rectangles accordingly with changing will it be full woth colour or not.

Cut tool allows you to take part of the drawing(rectangle 100 by 100) and then paste where you wish it to be, also will not delete object you draw. (to start cutting click select area)

Advanced stanp tool with circle, triangel, square, pentagon and star, with changeable fill, stroke, color of fill, color of stroke.

Text allows you to write text right after sign ">" wherever you on want on canvas.
